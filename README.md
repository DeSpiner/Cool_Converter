# Cool Converter

**Multi-purpose terminal-based converter written in Python.**

This project allows you to:

- Convert social media views to estimated earnings
- Convert money to in-game currencies for popular games
- Convert text to binary and back
- Calculate what you could have done with your screen time

---

## Key Features

- Interactive terminal interface with animated text
- Universal keybinds:
  - `Q` – Quit
  - `I` – Show Info
  - `M` – Main Menu
  - `B` – Go back to the previous menu
- Supports multiple platforms, video types, and currencies
- Handles user input errors gracefully
- Clear and readable code, structured for learning
- **AI assistance was used only to help write this README and to understand parts of the code, but all code was written solely by the author**
